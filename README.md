# 🚀 VaaniAI AdStudio

VaaniAI AdStudio is an AI-powered advertisement generation platform built using Google AI Studio that helps businesses create professional, platform-specific marketing creatives in seconds.

## 🔗 Live Demo
👉 Try it here: https://ais-dev-tykidxr2y2u5liielcybfj-779982671696.asia-southeast1.run.app/

## ✨ Overview

This project allows users to generate high-quality advertisements tailored for different platforms such as:
- Instagram Ads
- Facebook Ads
- WhatsApp Marketing
- Print / Digital Posters

The system ensures that each output is optimized for the selected platform format, helping businesses create ready-to-use marketing content instantly.

## 🎯 Key Features

- 🎨 AI-generated advertisement creatives
- 📱 Platform-specific formatting (Instagram, WhatsApp, Posters, etc.)
- ⚡ Instant ad generation
- 🧩 Simple and user-friendly interface
- 🚫 Focused on ads only (not generic content posts)

## 💡 Problem It Solves

Small and medium businesses often struggle with creating professional marketing creatives for different platforms. VaaniAI AdStudio solves this by automating ad creation using AI, saving both time and design effort.

## 🛠️ Tech Stack

- Google AI Studio
- Generative AI APIs
- Frontend: Vite
- Backend: Node.js

## 🚀 How It Works

1. User selects a platform (Instagram / WhatsApp / Poster etc.)
2. Enters business details and content
3. AI generates a ready-to-use advertisement
4. Output is optimized for selected platform format
